### Key Findings

1.  **Electric Fields Enhance Pattern Complexity**
    *   The model shows that incorporating an electrostatic field into a standard bioelectric network model increases the complexity of membrane voltage (`V_mem`) patterns.
    *   Complexity is measured using the **Tononi-Sporns-Edelman (TSE)** metric, which quantifies the balance between integration and segregation in a system.
    *   An optimal parameter region exists where pattern complexity is maximized, dependent on field sensitivity (`W` and `B`) and action range (`R`).

2.  **Field Acts as a Synergetic Control Parameter**
    *   The electric field exhibits properties consistent with **Haken's synergetics**: it operates on a slower timescale and with lower dimensionality than the cellular `V_mem` dynamics.
    *   The field effectively "enslaves" the faster `V_mem` dynamics, acting as a **macroscopic control parameter**. This is quantified by measures of **Relative Field Compression (RFC)** and **Field Influence (FI)**.

3.  **Field Catalyzes Long-Distance Causal Interactions**
    *   The model demonstrates that the field facilitates **non-local causal interactions** between cells, even beyond the immediate field action range.
    *   **Causal Strength (CS)** and **Causal Distance (CD)** are highest in models with strong field sensitivity, showing that the most influential interactions are not always between immediate neighbors.

### Model Description (Figure 10)

The core of the paper is a 2D computational model of an 11x11 cell sheet:
